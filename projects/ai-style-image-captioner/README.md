# AI-Style Image Captioner

A fun web application that allows users to upload images and generate creative, AI-style captions dynamically.

## Features

- Upload images (JPEG, PNG, etc.)
- Generate random creative captions using predefined templates
- Copy captions to clipboard for easy sharing
- Responsive design

## How It Works

The app uses a set of caption templates and fills them with random adjectives and nouns to create unique captions. While not using actual AI image analysis, it provides fun and varied results.

## Usage

1. Click "Choose File" to select an image
2. Once uploaded, click "Generate Caption"
3. Copy the generated caption using the "Copy Caption" button

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
