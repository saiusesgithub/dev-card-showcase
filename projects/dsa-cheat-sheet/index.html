ARRAY +

A contiguous block of memory storing elements of the same type.

Common Operations:

  • Access by index O(1)
  • Search (unsorted) O(n)
  • Insert at end O(1)
  • Insert at position O(n)
  • Delete at position O(n)

Time Complexity:

Operation Average Worst
Access O(1) O(1)
Search O(n) O(n)
Insertion O(n) O(n)
Deletion O(n) O(n)

Space Complexity: O(n)

LINKED LIST +

A linear collection of nodes, each containing data and a pointer to the next node.

Common Operations:

  • Access by index O(n)
  • Search O(n)
  • Insert at head O(1)
  • Delete at head O(1)
  • Insert/delete at position O(n)

Time Complexity:

Operation Average Worst
Access O(n) O(n)
Search O(n) O(n)
Insertion O(1) O(1)
Deletion O(1) O(1)

Space Complexity: O(n)

STACK +

LIFO (Last-In-First-Out) data structure.

Common Operations:

  • Push (add to top) O(1)
  • Pop (remove from top) O(1)
  • Peek (view top) O(1)
  • Search O(n)

Time Complexity:

Operation Average Worst
Push O(1) O(1)
Pop O(1) O(1)
Peek O(1) O(1)
Search O(n) O(n)

Space Complexity: O(n)

QUEUE +

FIFO (First-In-First-Out) data structure.

Common Operations:

  • Enqueue (add to back) O(1)
  • Dequeue (remove from front) O(1)
  • Peek (view front) O(1)
  • Search O(n)

Time Complexity:

Operation Average Worst
Enqueue O(1) O(1)
Dequeue O(1) O(1)
Peek O(1) O(1)
Search O(n) O(n)

Space Complexity: O(n)

TREE (Binary Search Tree) +

Hierarchical data structure with a root value and subtrees of children.

Common Operations:

  • Search O(log n)
  • Insert O(log n)
  • Delete O(log n)
  • Traversal O(n)

Time Complexity:

Operation Average Worst
Access O(log n) O(n)
Search O(log n) O(n)
Insertion O(log n) O(n)
Deletion O(log n) O(n)

Space Complexity: O(n)

GRAPH +

A non-linear data structure of nodes (vertices) and edges.

Common Operations:

  • Add vertex O(1)
  • Add edge O(1)
  • Remove vertex O(V+E)
  • Remove edge O(E)
  • Query O(V)

Time Complexity (Adjacency List):

Operation Average Worst
Add vertex O(1) O(1)
Add edge O(1) O(1)
Remove vertex O(V+E) O(V+E)
Remove edge O(E) O(E)

Space Complexity: O(V+E)

SORTING ALGORITHMS +

Algorithms to arrange elements in a specific order.

Common Algorithms:

Algorithm Best Average Worst Space
Bubble Sort O(n) O(n²) O(n²) O(1)
Selection Sort O(n²) O(n²) O(n²) O(1)
Insertion Sort O(n) O(n²) O(n²) O(1)
Merge Sort O(n log n) O(n log n) O(n log n) O(n)
Quick Sort O(n log n) O(n log n) O(n²) O(log n)
SEARCHING ALGORITHMS +

Algorithms to find an element within a data structure.

Common Algorithms:

Algorithm Best Average Worst Space
Linear Search O(1) O(n) O(n) O(1)
Binary Search O(1) O(log n) O(log n) O(1)
Depth-First Search O(V+E) O(V+E) O(V+E) O(V)
Breadth-First Search O(V+E) O(V+E) O(V+E) O(V)
