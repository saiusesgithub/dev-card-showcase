What is this project about?

Community Card Showcase is a beginner-friendly Open Source project where contributors add their own profile cards to showcase their participation in the open source community.

It's designed to help newcomers make their first contribution in a supportive environment with clear guidelines and helpful community members.

Why is my profile image not showing up?

Common image issues and their solutions:

  • Ensure the image is square (1:1 aspect ratio) for proper display
  • Only .jpg or .png formats are supported (no .gif, .webp, etc.)
  • Image name must match your GitHub username exactly (case-sensitive)
  • File should be placed inside the images/ folder
  • Maximum file size is 2MB - compress if necessary
  • Check file permissions and ensure it's not corrupted

Why is my contribution card not visible?

If your card isn't appearing after submission, check these common issues:

  • Card must be pasted inside the CONTRIBUTORS section in index.html
  • Check for missing or mismatched HTML tags (div, class, etc.)
  • Make sure the image path is correct (should be images/username.png)
  • Reload the page without cache (Ctrl+F5 or Cmd+Shift+R)
  • Wait a few minutes after merging - GitHub Pages takes time to update
  • Verify your PR has been successfully merged, not just submitted

Common PR mistakes to avoid

These mistakes can delay or prevent your PR from being merged:

  • Editing other contributors' cards or code without permission
  • Uploading large, non-square, or inappropriate images
  • Making multiple unrelated changes in one PR
  • Creating PR from the main branch instead of a feature branch
  • Not following the template structure exactly
  • Forgetting to test your changes locally first
  • Ignoring the contribution guidelines

Where can I get help with my contribution?

If you're stuck or confused, here's how to get assistance:

  • Re-check the Contribute page for detailed instructions
  • Review the Guidelines page for community standards
  • Open a GitHub issue with clear screenshots and description
  • Ask politely in the issue comments - our community is friendly!
  • Check if someone else has already asked a similar question
  • Look for existing PRs to understand the expected format

Our maintainers typically respond within 24-48 hours.

How long does PR review usually take?

Our maintainers are volunteers who review PRs in their free time. Typical review times are:

  • 1-3 days for simple card additions
  • 3-7 days for code changes or complex contributions
  • Longer during holidays or busy periods

Please be patient and don't tag maintainers unless it's been over a week without response.

Can I customize my profile card?

Yes, within reasonable limits! You can:

  • Choose your role badge color (from predefined options)
  • Add up to 3 technology icons to showcase your skills
  • Include a short bio (max 150 characters)
  • Link to your portfolio, blog, or other relevant sites
  • Select a theme color that matches your personal brand

Please don't modify the card structure or add custom CSS/JavaScript to your individual card.
